# Pirâmide de testes

# O que é
Distribui a quatidade necessária dos testes em 3 tipo:

- Testes unitários
- Testes de integração
- Testes END-TO-END (e2e)

# Testes Unitários - Base
Testes isolados que focam em apenas uma unidade por vez.

# Testes de Integração - Meio
Testes que focam em verificar a integração de partes do projeto.

# Testes END-TO-END - Topo
Testes que focam em verificar o bom funcionamento da aplicação como um usuário final.
