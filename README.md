# 🚀 Gerenciador de Tarefas: Angular Test & Quality
Repositório dedicado ao estudo e implementação de estratégias avançadas de testes em ecossistemas Angular, focado em estabilidade, manutenibilidade e confiança do código.

# O que vai conter nesse projeto
- Testes Unitários 
- Testes de Integração 
- End-to-End (E2E)
- Técnicas de teste

# O que vai ser criado
Vamos criar uma aplicação que gerencia tarefas com todas as operações essenciais!

# Tecnologias utilizadas
- Angular 18
- NX
- Jest
- Cypress
- Ng-mocks
- TailwindCSS

# Técnicas 
- Conventional Commits
- TDD
- Pirâmide de Testes
- Testes resilientes
- Mocking
- Smart / Dumb Components
